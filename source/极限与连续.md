# 函数极限

## 概念题

函数 f(x)在 x=x0处连续，指的是：

> limx→x0f(x)=f(x0)，且 f(x0)存在，且极限存在并相等。

**它只保证“在 x0这个点”满足连续条件，不保证在它周围的点（哪怕很近）也连续！**只针对点，可以联系可去间断点





极限存在的前提之一是：**在某个去心邻域内，函数必须处处有定义。**



函数极限的保号性

设 x→x0limf(x)=A（同理可推广到 x→∞）。

- **正向**：若 A>0（或 A<0），则存在 δ>0，当 0<∣x−x0∣<δ时恒有 f(x)>0（或 f(x)<0）。这里正向不能等于
- **逆向**：若在 x0某去心邻域内恒有 f(x)≥0（或 f(x)≤0），且极限存在，则 A≥0（或 A≤0）。若条件是 “f(x) > 0”，→ 结论仍是 “A ≥ 0”



极限反映的是“趋近于某点时的行为”，不是“某点处的值”，更不是“整个区间上的值”。因此：

- “极限大” ⇒ “在去心邻域内函数值大” —— **这是可以严格推导的（保号性）**
- “邻域内函数值大” ⇒ “极限大” —— **只能推出“≥”，不能推出“>”**即使邻域里函数值一直严格大于，它们也可以“无限逼近同一个数”，导致极限相等。



极限的存在性，题目没说极限等于一个数就不一定极限存在

### 巧妙方法

垂直渐近线

找出函数图像在垂直方向“无限逼近但不接触”的直线。

**操作口诀**：

> “找无定义点 → 看极限是否为∞(可能从左侧右侧逼近)”



水平渐近线

x趋近于无穷算极限是不是数字

这里一定要注意，我们球的是趋于无穷的两种情况一种是趋于整的一种是趋于负的，比如说e的x次方这种不考虑好了就会出问题

若已存在水平渐近线，通常无斜渐近线。



斜渐近线



在证明的时候用基本方法的定义求(斜率为0或者不存在就不会有斜渐近线)，在已经知道存在的时候用特殊方法求



把 f(x) **强行拆成 “直线部分 + 无穷小部分”** → 直线部分就是渐近线！

f(x) = αx + β + γ(x)，其中

- α, β 为常数，
- γ(x) → 0（当 x→∞），→ 则 **y = αx + β 就是渐近线**！

这里一定要注意，展开对象是“趋于0的小量”（如 x1,x1,x−a等）

$(1+x)^\alpha = x^\alpha \left(1 + \frac{1}{x}\right)^\alpha$

假如遇到这种趋近无穷的要先做代换

# 泰勒多项式

f(x)=n=0∑∞n!f(n)(a)(x−a)n

常说的泰勒展开公式，其实大多是**麦克劳林展开**，也就是在 x=0处的泰勒展开。

麦克劳林展开要求展开点附近变量趋于 **0**，因此**只能在无穷小量上使用**。

例如 $(1+x)^{1/x}$，当 x→∞时，不能直接套用麦克劳林展开式，因为此时 x不是无穷小，且指数中也含有 x。

正确做法是先**取对数**，将其化为 exp(x1ln(1+x))，再把 x1视为无穷小量进行展开，最后再还原指数。
