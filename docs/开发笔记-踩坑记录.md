# 开发笔记与踩坑记录

本文档记录从桌面笔记 → 墨墨/Anki 卡组过程中遇到的实际问题与解决方案，便于换电脑或他人复现时少走弯路。

---

## 1. 墨墨记忆卡 vs Anki

| 问题 | 墨墨记忆卡 | Anki |
|------|-----------|------|
| LaTeX | 客户端对 `\( … \)` 支持不稳定，公式常显示为源码 | 2.1+ 内置 MathJax，直接渲染 |
| 可行方案 | 用 `math_to_image.py` 把含 LaTeX 的整行渲染成 PNG 嵌入 HTML | 保留 LaTeX 字符串即可 |
| 包体积 | 图片版约 3.6 MB（249 张 PNG） | 纯文本约几十 KB |
| 维护成本 | matplotlib 兼容性、字体、换行都要调 | 低 |

**结论**：最终只维护 **Anki 版**（`generate_math_review_apkg.py` 默认输出）。墨墨图片管线（`math_to_image.py` + `_apkg_media/`）保留作参考，不纳入 git（缓存目录已 `.gitignore`）。

### 墨墨图片版踩坑（matplotlib）

以下 LaTeX 写法在 matplotlib mathtext 中会失败，需在 `math_to_image.py` 里预处理：

- `\frac1n` → `\frac{1}{n}`
- `\frac\pi2` → `\frac{\pi}{2}`
- `\iff` → `\Leftrightarrow`
- `\sqrt[n]{a}` 等 n 次根号

渲染策略：**按 `<br>` 分行，含公式的整行合成一张 PNG**，避免中文与公式混排错位。

## 2. 卡片粒度：155 张 → 45 张

初版按笔记每条要点一张卡，共 **155 张**，刷起来太碎、上下文丢失。

合并原则：

- **公式表**整表一张（二倍角、降幂、泰勒各节、黎曼和取点表）
- **方法列表**整段一张（0/0 型速查、「一看就别积分」）
- **同一小节要点**一张（奇偶性 3 条、周期性 4 条）
- **不同概念**仍分开（保号性 vs 其他要点）

## 3. 公式卡 vs 方法卡

| 类型 | 背面 | 例题 |
|------|------|------|
| 公式卡 | 只给公式/填空答案 | 无 |
| 方法/概念卡 | 答案 + **例（方法）** | 演示「遇到这类题怎么走」 |

曾误给公式卡也加例题，后改为 `method_example=` 参数仅用于方法卡。

## 4. 填空难度：避免「猜一个词」

早期正面如「极限不存在时，______」——背面可能是「无界 / 不能 / 端点极限」之一，只能靠猜。

改进：

1. 编号 + **【情境标签】**：`①【端点取不到】…`
2. 判断类改成**问句**：「能否直接断定无界？答：______」
3. 背面也带编号，与正面对应

## 5. 笔记与卡片勘误（重要）

对照笔记逐条核对时发现并已修正：

### 5.1 高阶无穷小定义写反

- **错**：`\lim \beta/\alpha = 0` ⇒ **α** 是 β 的高阶无穷小
- **对**：⇒ **β** 是 α 的高阶无穷小（β = o(α)）

### 5.2 有界性反例不当

- **错**：用 `sin x / x` 说明「极限不存在不能判无界」（其实该极限 = 1，存在）
- **对**：`sin(1/x)` 在 x→0 极限不存在，但在去心邻域内有界

### 5.3 周期性结论缺前提

- **错**：积分在一个周期内为零 ⇔ 原函数有相同周期（无条件）
- **对**：需 **f 以 T 为周期且可积**；反例：`sin²x` 周期 π，但原函数 `-cos²x/2` 周期 2π

### 5.4 绝对值三角不等式

- 恒成立：`| |a_n| - |a| | ≤ |a_n - a|`
- `< |a|/2` 只在证明中**额外假设** `|a_n - a| < |a|/2` 时使用，不能写进恒成立式

### 5.5 其他

- 施托尔兹推广：加条件 `x_n > 0`、右端极限存在
- 无穷大表：区分 `x sin x`（无界）与 `sin x/x`（有界）
- 第二条 sin 不等式、黎曼和 `i/n` 系数须为 1、海涅定理完整式等已补入卡片

## 6. Anki 安装（Windows）

`winget install Anki.Anki` 可能显示成功但 `Program Files` 下找不到 `anki.exe`。

实际安装路径常为：

```
%LOCALAPPDATA%\Programs\Anki\Anki.exe
```

首次启动需完成配置文件创建，再导入 `.apkg`。无官方 CLI 静默导入，需 GUI 确认或安装 AnkiConnect 插件。

---

## 7. genanki 警告

字段中含未转义的 `<`（如 `$0 < x < \pi/2$`）会触发 HTML 警告。Anki 端通常仍能显示；若要消除警告，需对非 LaTeX 部分的 `<` 做 `html.escape()`。

## 8. 掌握度自评（仅供参考）

- 背要点 / 概念 / 易错：**85–90%**
- 极限大题选对方法：**70–80%**（需配合刷题）
- 独立算对：**50–65%**

卡组适合**要点回忆与方法选型**，不能替代完整刷题。
