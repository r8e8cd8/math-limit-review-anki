## 函数



### 奇偶性

- 偶函数的原函数**不一定是**奇函数；只有在积分区间为 $[0, x]$ 时，偶函数的原函数才是奇函数。  
（速记：**哦!飞机**）
- 可通过求导处理含积分的函数，但要注意：函数的原函数符号与预期**相反**。
- 无从下手时，回到奇偶性的定义来分析。



### 周期性

- **核心结论**：若 \(f\) 以 \(T\) 为周期且可积，则 \(\displaystyle\int_0^T f = 0 \Leftrightarrow\) 其原函数也以 \(T\) 为周期。
- 处理嵌套分段函数时，先分析**外层函数**：内层取不同值，对应外层不同分段。
- 周期函数在周期内图像相同，但整体是平移关系，可用 $y = f(ax)$ 来理解。
- 计算困难时，回到周期函数的定义。



### 有界性

- 函数在区间端点处**取不到**时，必须考察端点的极限。
- 若存在子列使函数值趋于无穷，则函数在该点附近**无界**。
- 极限不存在，**不能**直接判定无界。例如 $\sin\dfrac{1}{x}$ 在 $x \to 0$ 时极限不存在，但在去心邻域内有界。

## 连续与间断点



### 连续函数的四则运算


| 运算  | 结论                      | 注意                    |
| --- | ----------------------- | --------------------- |
| 加法  | $f(x) \pm g(x)$ 连续      | ✅                     |
| 乘法  | $f(x) \cdot g(x)$ 连续    | ✅                     |
| 除法  | $\dfrac{f(x)}{g(x)}$ 连续 | ⚠️ 要求 $g(x_0) \neq 0$ |
| 常数倍 | $C \cdot f(x)$ 连续       | ✅（$C$ 为常数）            |




### 间断点分类


| 类型    | 形象比喻    | 关键特征            |
| ----- | ------- | --------------- |
| 可去间断点 | 断桥（缺一块） | 左右极限相等，但不等于函数值  |
| 跳跃间断点 | 台阶 / 断层 | 左右极限不等          |
| 无穷间断点 | 万丈深渊    | 极限为 $\pm\infty$ |
| 振荡间断点 | 暴风吊桥    | 极限不存在，也不趋于无穷    |

## 函数极限



### 概念题

**连续的定义**

函数 $f(x)$ 在 $x = x_0$ 处连续，指的是：

$$\lim_{x \to x_0} f(x) = f(x_0)$$

且 $f(x_0)$ 存在，且极限存在并相等。

> 连续**只保证在 $x_0$ 这一点**满足条件，不保证周围点也连续。可联系**可去间断点**理解。

**极限存在的前提**

在某个**去心邻域**内，函数必须处处有定义。

**保号性**

设 $\displaystyle\lim_{x \to x_0} f(x) = A$（同理可推广到 $x \to \infty$）。


| 方向  | 条件                                       | 结论                       |
| --- | ---------------------------------------- | ------------------------ |
| 正向  | $A > 0$（或 $A < 0$）                       | 存在 $\delta > 0$，当 $0 <   |
| 逆向  | 在去心邻域内恒有 $f(x) \geq 0$（或 $\leq 0$），且极限存在 | $A \geq 0$（或 $A \leq 0$） |


> 注意：逆向中即使 $f(x) > 0$ 严格成立，结论仍是 $A \geq 0$，不能推出 $A > 0$。

**其他要点**

- 极限反映的是"趋近时的行为"，不是某点处的值，更不是整个区间上的值。
- "极限大" $\Rightarrow$ 去心邻域内函数值大（保号性，可严格推导）。
- "邻域内函数值大" $\Rightarrow$ 只能推出极限 $\geq$，不能推出 $>$。
- 题目未说明极限等于某数时，**不一定**极限存在。
- 判断 $a_n$ 与 $a$ 的关系时，常用定义式，构造合适的 $\varepsilon$。

### 重要极限

$$\lim_{n \to \infty} \left(1 + \frac{a}{n}\right)^n = e^a \quad \text{（特别地，} a = 1 \text{ 时得 } e \text{）}$$

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

### $\dfrac{0}{0}$ 型

**主要工具**：泰勒公式、等价无穷小。


| 情形                 | 方法                                |
| ------------------ | --------------------------------- |
| 分母简单（如 $2x$）       | 洛必达法则；导数难求时用对数求导                  |
| 根号减根号              | 乘以共轭；根号减无根号项也可尝试                  |
| $\ln(1 + f(x))$ 展开 | 至少展开到 $\dfrac{f(x)^2}{2}$，否则可能丢精度 |
| $x^x$ 型            | 先提 $x$，再用 $e^{x\ln x}$ 代换         |
| 反三角函数              | 记住常用展开公式                          |
| 拉格朗日中值定理型          | 通过夹住两个函数确定中值项大小；能代入则代入，否则等价替换     |

### $\dfrac{\infty}{\infty}$ 型

- **抓大头**：注意不能对 $e^x$ 这类函数简单抓大头——底数中的微小扰动可能通过指数产生**非线性放大**。
- 也可用拉格朗日中值定理（可能给 $f(0)$ 等具体值让你自己找）。
- **倒代换**：令 $t = \dfrac{1}{x}$ 等。
- **海涅定理**：令 $x = \dfrac{1}{n}$（必须如此），则 $n \to \infty$ 时 $x \to 0^+$，可将离散数列极限 $\displaystyle\lim_{n \to \infty} f\left(\dfrac{1}{n}\right)$ 转化为连续极限 $\displaystyle\lim_{x \to 0^+} f(x)$。
- $\sqrt{x^2} = |x|$，注意符号。

---



### $\infty - \infty$ 型

- 主要方法：**代换**、根式有理化、通分。
- 可能需提项（提出根号内部分子，或处理无分母情形）。

---



### $0^0$ 与 $\infty^0$ 型

恒等变形：$u^v = e^{v \ln u}$。

---



### $1^\infty$ 型

化为 $e^{v(u-1)}$ 形式（即指数乘上"底数减 1"）。

### 含 $e$ 或绝对值

- 绝对值：**分情况讨论**。
- 经典极限：

$$\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n = e$$

$$\lim_{x \to 0} (1 + x)^{\frac{1}{x}} = e$$

---



### 用导数定义求极限

常见形式：$\dfrac{f(x+a) - f(x)}{a}$，其中 $a$ 可能起迷惑作用。

- 关键：根据此式凑出导数定义，解出已知数值的 $f'(x)$。
- 也可能出现 $f(x+a) - f(x+b)$，需分开配凑；或除以 $f(\text{某式})$ 再右边乘回。
- **不能**单独对小部分用，必须整体凑定义。
- 判断一点可导：左右导数存在且相等，极限须从**双侧**逼近且结果一致。

---



### 三角处理

$$\sin(x - n\pi) = (-1)^n \sin x$$

$$\cos(x - n\pi) = (-1)^n \cos x$$

$$\tan(x - n\pi) = \tan x$$

---



### 积分形式极限

- 被积函数含多参数时换元，换元不让 $x$ 变，变换上下限。
- 尝试提出 $e^x$，或直接在积分内部代入；被夹住的中值 $\xi$，记得代入如 $e^0$、$\cos 0$ 等。
- **积分中值定理**：$\displaystyle\int_a^b f(x)dx = f(\xi)(b-a)$（此处掌握需加强）。
- 积分上限趋于 0，被积函数在 0 附近等价 $\Rightarrow$ 积分也等价。
- **二重变限积分**：交换积分次序，把最内层变量提出来。
- **二重积分中值定理**：$\displaystyle\iint_D f(x,y)d\sigma = f(\xi,\eta) \cdot S_D$。

## 数列极限



### 以积分形式给出



#### 可积可化简

直接化简后求极限。

#### 不可积不可化简

采用**积分中值定理**，并用上下限逼近（数列中 $n \to +\infty$，或构造积分上下限对应的放缩）。

**「一看就别积分」速查表**


| 判定特征           | 典型例题                                                                   | 正确操作（❌ 别积分）                                                                |
| -------------- | ---------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| 含 $x^n$        | $\displaystyle\lim_{n \to \infty} \int_0^1 \frac{x^n}{1+x}dx$          | 积分中值定理：$\dfrac{\xi^n}{1+\xi} \to 0$                                        |
| 含 $e^{\pm nx}$ | $\displaystyle\lim_{n \to \infty} \int_0^1 e^{-nx}\cos xdx$            | 积分中值定理：$e^{-n\xi}\cos\xi \to 0$                                            |
| 含 $\sin^n x$   | $\displaystyle\lim_{n \to \infty} \int_0^{\pi/2} \sin^n xdx$           | 积分中值定理：$\sin^n\xi \cdot \dfrac{\pi}{2} \to 0$                              |
| 根号 + 幂次        | $\displaystyle\lim_{n \to \infty} \int_0^1 \frac{x^n}{\sqrt{1+x^2}}dx$ | 放缩：$0 \leq \cdots \leq \displaystyle\int_0^1 x^ndx = \dfrac{1}{n+1} \to 0$ |
| 参数嵌分母          | $\displaystyle\lim_{n \to \infty} \int_0^1 \frac{\ln(1+x)}{1+x^n}dx$   | 放缩 + 分段估计（极限 $= \ln 2$）                                                    |


---



### 以 $n$ 项和形式给出



#### 两个经典数列极限及不等式

$$\lim_{n \to \infty} \sqrt[n]{a} = 1 \quad (a > 0)$$

$$\lim_{n \to \infty} \sqrt[n]{n} = 1$$

$$\sin x < x < \tan x \quad \left(0 < x < \frac{\pi}{2}\right)$$

$$\frac{2}{\pi}x < \sin x < x < \frac{\pi}{2}\sin x \quad \left(0 < x < \frac{\pi}{2}\right)$$

$$e^x > 1 + x \quad (x \neq 0)$$

$$\frac{x}{1+x} < \ln(1+x) < x \quad (x > -1,\ x \neq 0)$$

#### 变成一重积分（黎曼和）

1. 凑出 $\dfrac{i}{n}$（$i = 1, 2, \ldots, n$）。
2. 替换求和起止元素为 $1$ 和 $m$（有限项不影响极限）。
3. 最后面的 $\dfrac{x}{n}$ 就是积分区间 $[a,b]$ 的每一小段。


| 取点 $\xi_i$         | Riemann 和形式                                                                     | 对应分割           |
| ------------------ | ------------------------------------------------------------------------------- | -------------- |
| $\dfrac{i}{n}$     | $\displaystyle\sum_{i=1}^{n} f\left(\dfrac{i}{n}\right) \cdot \dfrac{1}{n}$     | 右端点，$[0,1]$ 等分 |
| $\dfrac{i-1}{n}$   | $\displaystyle\sum_{i=1}^{n} f\left(\dfrac{i-1}{n}\right) \cdot \dfrac{1}{n}$   | 左端点            |
| $\dfrac{2i-1}{2n}$ | $\displaystyle\sum_{i=1}^{n} f\left(\dfrac{2i-1}{2n}\right) \cdot \dfrac{1}{n}$ | 中点             |


> **关键结论**：$f$ 在 $[a,b]$ 上可积（通常连续即可），左端点、右端点、中点取点得到的极限**完全相同**，都等于 $\displaystyle\int_a^b f(x)dx$。  
> 需要 $\dfrac{i}{n}$ 的系数为 $1$ 才能保证一一对应。



#### 夹逼定理

先放缩，再用上述黎曼和处理。

**常用技巧**

- 在原分母的 $i^2$ 前（**不能动 $n^2$，积分值会变**）配待定系数 $c_n$，令 $\dfrac{i}{n^2 + n + c_n i^2} = k_n \cdot \dfrac{i}{n^2 + i^2}$。
- 或变成 $i + a$ 从而消去 $\dfrac{i + k}{n}$ 中的无关系数 $k$。
- 主动构造：

$$1 = \sum_{k=1}^{n} \frac{1}{n}$$

$$1 - \frac{1}{n+1} = \sum_{k=1}^{n} \frac{1}{k(k+1)}$$

$$\frac{1}{3} = \sum_{k=1}^{n} \left(\frac{k}{n}\right)^2 \cdot \frac{1}{n}$$

#### 变成二重积分

提出 $\dfrac{1}{n^2}$，积分区间变化后得到 $j$ 和 $i$ 的等式，把 $y$ 代入 $\dfrac{j}{n}$、$x$ 代入 $\dfrac{i}{n}$，得到 $x$ 与 $y$ 的关系（积分区域）。

---



### 以 $n$ 次根号或乘积形式给出

- $n$ 次根号下 $n$ 次方的值 $\to$ **最大值**；无穷多项时也找最大值，可能用单调性。
- 乘积取对数 $\to$ 变成求和。

---



### 递推数列求极限（单调有界）



#### 等价代换

$$n! \sim \sqrt{2\pi n}\left(\frac{n}{e}\right)^n \sim \frac{\sqrt{2\pi n}}{e^n} \cdot n^n$$

$$(\sqrt{2\pi n})^{1/n} \to 1 \quad (n \to \infty)$$

#### 施托尔兹定理（数列版洛必达）

$$\lim_{n \to \infty} \frac{a_n}{b_n} = \lim_{n \to \infty} \frac{a_{n+1} - a_n}{b_{n+1} - b_n} \quad (b_n \nearrow +\infty)$$

**推广**

$$\lim_{n \to \infty} \sqrt[n]{x_n} = \lim_{n \to \infty} \frac{x_{n+1}}{x_n} \quad (x_n > 0,\ \text{右端极限存在})$$

#### 存在性

- **单调有界必存在**（单调性若求导不好证，可用定义证明）。
- 对 $x_{n+1} = f(x_n)$，若收敛，极限 $a$ 必满足 $a = f(a)$，解方程并结合实际意义筛选。

**例**：$x_{n+1} = \sqrt{6 + x_n}$，设极限为 $a$，由连续性得 $a = \sqrt{6 + a}$，解得 $a = 3$ 或 $-2$，由根号非负舍去 $-2$，故极限为 $3$。

## 渐近线



### 基本方法（定义法）

**斜率 $k$**

$$k = \lim_{x \to \infty} \frac{f(x)}{x}$$

> 把函数"拉平"成一次函数后，一次项系数就是斜率。

**截距 $b$**

$$b = \lim_{x \to \infty} \bigl[f(x) - kx\bigr]$$

斜渐近线：$y = kx + b$。

---



### 垂直渐近线

找出图像在垂直方向"无限逼近但不接触"的直线。

> **口诀**：找无定义点 → 看极限是否为 $\infty$（可能从左右两侧逼近）。

---



### 水平渐近线

计算 $x \to +\infty$ 与 $x \to -\infty$ 的极限是否为有限数。

> 注意：必须分别考虑 $+\infty$ 和 $-\infty$，如 $e^x$ 两侧行为不同。  
> 若已存在水平渐近线，通常**无**斜渐近线。

---



### 斜渐近线（特殊方法）

**证明时**用定义法（斜率为 0 或不存在则无斜渐近线）；**已知存在时**用拆分法。

把 $f(x)$ **强行拆成"直线部分 + 无穷小部分"**：

$$f(x) = \alpha x + \beta + \gamma(x)$$

其中 $\alpha, \beta$ 为常数，$\gamma(x) \to 0$（$x \to \infty$），则 $y = \alpha x + \beta$ 就是斜渐近线。

> 展开对象是**趋于 0 的小量**（如 $\dfrac{1}{x}$、$x - a$ 等）。

**例**：$(1+x)^\alpha = x^\alpha \left(1 + \dfrac{1}{x}\right)^\alpha$，遇 $x \to \infty$ 时先做代换。

**简单有理函数**：做多项式长除法，拆成「一次多项式 + 趋于 0 的余项」，丢掉余项即得斜渐近线。

## 极限等式

若极限存在，则极限值 $A$ 是一个**确定的实数**（可用于反推未知参数）。

## 无穷小

$$f(x) = o(x^k) \quad (x \to 0) \iff \lim_{x \to 0} \frac{f(x)}{x^k} = 0$$

> 要严格用此式证明，否则容易出错。

- 若 $\displaystyle\lim \frac{\beta}{\alpha} = 0$，则 $\beta$ 是 $\alpha$ 的**高阶无穷小**。  
记忆：$x^3 / x^2 \to 0$，$x^3$ 是 $x^2$ 的高阶无穷小。
- **常数列 0 是无穷小**：$n \to \infty$ 时极限为 0 的数列（或函数）称为无穷小。

---



## 无穷大


| 函数                  | 图像特征       |
| ------------------- | ---------- |
| $x\sin x$           | 循环往复的"扩音器"（可取特殊数列说明**无界**或极限不存在） |
| $\dfrac{\sin x}{x}$ | 循环往复的"缩音器"（**有界**；$x\to\infty$ 时极限为 0） |


关键：对 $x\sin x$ 等，取特殊数列使其经过 $\sin x = 0$ 的点，从而说明无界或极限不存在。

## 泰勒多项式

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$

常说的泰勒展开，大多是**麦克劳林展开**（在 $x = 0$ 处的泰勒展开）。

**使用条件**

- 麦克劳林展开要求展开点附近变量趋于 **0**，因此**只能在无穷小量上使用**。
- 例：$(1+x)^{1/x}$ 当 $x \to \infty$ 时，不能直接套用麦克劳林展开（$x$ 不是无穷小，指数中也含 $x$）。
- **正确做法**：先取对数化为 $\exp\left(\dfrac{\ln(1+x)}{x}\right)$，把 $\dfrac{1}{x}$ 视为无穷小量展开，再还原指数。
